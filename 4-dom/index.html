Exercise 4: DOM

Instructions: You can also create elements using Javascript. For this exercise we're going to do some simple DOM creation. Given an array of strings, you should render a bulleted list within the element .output.

Challenge: Once you've finished the task, try creating more than just text in the list... perhaps some checkboxes and labels to tick off the list?

Tip: DOM stands for Document Object Model and it's more than just HTML elements.
